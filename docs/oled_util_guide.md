# 圖片轉換幫手

## Installation

1. 從github下載pre-compile binary
2. 從source file 直接安裝

## 從Binary下載安裝

## 從source file 安裝

```
git clone https://github.com/EugenePY/model-oled-pb.git 
```

=== "OSX/Terminal"
    ```
    pip install ./ -e
    ```

=== "Windows/CMD" 
    ```
    pip install ./ -e
```

### 測試是否安裝完成

=== "OSX/Terminal"
    ```
    oled-utils --help
    ```

=== "Windows/CMD" 
    ```
    oled-utils --help

    ```
